Salon Analytics Project (30th May 2026)

A SQL Server project built using real salon transaction data.

Business Questions:

- Which payment method generates the most revenue?
- Which nail technician generates the most sales?
- What are the busiest days of the week?
- Who are the most loyal(visit most frequently) clients?
- Who are the most paying clients
- What services generate the highest revenue?
- Which is/are the slowest days in a month?
- Which nail technician generates the most tips?
- How much does the owner spend as perks every quater (in a year)?

Tools:

- SQL Server 2022
- Docker
- DataGrip
- Git
- GitHub
